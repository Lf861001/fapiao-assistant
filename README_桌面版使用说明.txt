发票递交助手 - 桌面版使用说明
1. 双击“发票递交助手.exe”启动。
2. 程序会自动打开浏览器页面。
3. 按合同填写公司代码、供应商代码、合同号/PO。
4. 点击“添加发票附件”和“添加送货单附件”逐次添加 PDF 文件，也可以把 PDF 直接拖入对应附件区域。
5. 每个发票 PDF 的文件名中必须包含一段连续 20 位数字，系统会将这 20 位数字提取为发票号码。
6. 点击“生成递交 PDF”后，本合同信息会锁定；需要点击“下一个合同”才能继续录入下一份。
7. 生成的 PDF 会保存在 exe 同目录下的 output 文件夹，也可以在页面右侧单独下载或批量下载 ZIP。
8. 每次启动工具时会自动清空 output 文件夹，请先保存好上一次需要保留的文件。

命名示例：
发票 PDF 文件名：invoice_12345678901234567890.pdf
生成结果：1219_855212_10594472_12345678901234567890.pdf
