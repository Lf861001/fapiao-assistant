发票递交助手 - 桌面版使用说明

1. 双击“发票递交助手.exe”启动。
2. 程序会自动打开浏览器页面。
3. 按合同填写公司代码、供应商代码、合同号/PO、发票号。
4. 点击“添加发票附件”和“添加送货单附件”逐次追加 PDF 文件，也可以把 PDF 直接拖入对应附件区域。
5. 发票附件文件名必须包含至少一个已填写的发票号，已添加附件可点击 X 删除。
6. 发票号数量必须和添加的发票 PDF 数量一致。
7. 点击“生成递交 PDF”后，本合同信息会锁定；需要点击“下一个合同”才能继续录入下一份。
8. 生成的 PDF 会保存在 exe 同目录下的 output 文件夹，也可以在页面右侧单独下载或批量下载 ZIP。
9. 每次启动工具时会自动清空 output 文件夹，请先保存好上一次需要保留的文件。

命名示例：
单张发票：1219_855212_10594472_65607584.pdf
连续发票：1219_855212_10594472_65607584-86&65607588-89.pdf
